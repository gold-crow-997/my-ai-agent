# Chat Agent

## About

Chat Agent is an AI-powered chatbot workflow for n8n that interacts with users via chat messages to provide company-specific responses and automate meeting scheduling using Google Calendar integration. It leverages OpenAI's GPT-4.1 model and maintains conversational context to deliver personalized, timely replies.

For setup, workflow details, configuration, and usage instructions, please see the [Full Documentation](https://kb.nmscreative.com/books/ai-agents/page/Chat-Agent).
