# CV Analysis
## About

This AI Agent automates the analysis of CVs submitted as PDFs or images, extracting and evaluating skills, experience, and qualifications against specified job criteria. It provides suitability ratings and concise summaries, streamlining recruitment processes.

For setup, workflow details, configuration, and usage instructions, please see the [Full Documentation](https://kb.nmscreative.com/books/ai-agents/page/cv-analysis)
